snowDetection(ref, tests, log)
Detects in multiple test videos the snowfall rate and
if there is snow on the road or not. Uses a reference video to
test every other videos.
 
Args:
    ref (str): path to reference video
    tests ([str]): list of paths to test videos
    log (bool): indicates if a csv logfile must be made for this test
 
Returns:
    [(bool, float)]: list of tuples containing test results in the following format :
    [(snowOnRoad1, snowfallRate1), (snowOnRoad2, snowfallRate2),...]
